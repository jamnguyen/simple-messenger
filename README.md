# Simple Messenger

A simple application to demostrate how Websocket work.

## Stack

| Side   | Package                    |
| ------ | -------------------------- |
| Server | Node + Express + Socket.io |
| Client | React + Parcel + Socket.io |

## Development

Install dependencies:

```sh
npm i
```

Start local server:

```sh
npm run start:server
```

Start local client:

```sh
npm run start:client
```
